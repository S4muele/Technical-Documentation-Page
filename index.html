Introduction

Before the Flexbox Layout module, there were four layout modes:

  • Block, for sections in a webpage
  • Inline, for text
  • Table, for two-dimensional table data
  • Positioned, for explicit position of an element

The Flexible Box Layout Module, makes it easier to design flexible responsive layout structure without using float or positioning.

display: flex

Placing the CSS property display: flex on an element allows you to use other flex properties to build a responsive page.

.container {

display: flex;

}

flex-direction

The flex-direction property specifies the direction of the flexible items.

This example sets the direction of the flexible items inside the <div> element in reverse order:

div {

display: flex;

flex-direction: row-reverse;

}

Note: If the element is not a flexible item, the flex-direction property has no effect.

justify-content

The justify-content property aligns the flexible container's items when the items do not use all available space on the main-axis (horizontally).

This example aligns the flex items at the center of the container:

.container {

display: flex;

justify-content: center;

}

Tip: Use the align-items property to align the items vertically.

align-items

The align-items property specifies the default alignment for items inside the flexible container.

This code snippet centers the alignments for all the items of the flexible <div> element:

div {

display: flex;

align-items: center;

}

Tip: Use the align-self property of each item to override the align-items property.

flex-wrap

The flex-wrap property specifies whether the flexible items should wrap or not.

Use this syntax to make the flexible items wrap if necessary:

div {

display: flex;

flex-wrap: wrap;

}

Note: If the elements are not flexible items, the flex-wrap property has no effect.

flex-shrink

The flex-shrink property specifies how the item will shrink relative to the rest of the flexible items inside the same container.

This code snippet makes the second flex-item shrink three times more than the rest:

div:nth-of-type(2) {

flex-shrink: 3;

}

Tip: If the element is not a flexible item, the flex-shrink property has no effect.

flex-grow

The flex-grow property specifies how much the item will grow relative to the rest of the flexible items inside the same container.

The code in the following example lets the second flex-item grow three times wider than the rest:

div:nth-of-type(1) {flex-grow: 1;}

div:nth-of-type(2) {flex-grow: 3;}

div:nth-of-type(3) {flex-grow: 1;}

Tip: If the element is not a flexible item, the flex-grow property has no effect.

flex-basis

The flex-basis property specifies the initial length of a flexible item.

Here's how you set the initial length of the second flex-item to 100 pixels:

div:nth-of-type(2) {

flex-basis: 100px;

}

Note: If the element is not a flexible item, the flex-basis property has no effect.

flex-order

The order property specifies the order of a flexible item relative to the rest of the flexible items inside the same container.

The following code sets the order of the flexible items:

div#myRedDIV {order: 2;}

div#myBlueDIV {order: 4;}

div#myGreenDIV {order: 3;}

div#myPinkDIV {order: 1;}

Note: If the element is not a flexible item, the order property has no effect.

align-self

The align-self property overrides the flexible container's align-items property.

Center the alignments for one of the items inside a flexible element:

#myBlueDiv {

align-self: center;

}

Note: The align-self property overrides the flexible container's align-items property.
